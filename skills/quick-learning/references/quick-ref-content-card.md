# Quick Reference — Content Card

1. When placing a PNG with removed background (product shot) on a CSS platform surface -> calculate img.bottom = platform_top_px - transparent_space_bottom_px (Seen: 1)
2. When multiple text blocks on a card have different positions -> check visual weight to confirm intended reading order (Seen: 1)
3. When placing a text block on a full-bleed photo with uneven background -> align to least visually loaded zone (Seen: 1)
4. When marketplace photo has unwanted background -> run rembg BEFORE generating HTML to avoid photo brick in design (Seen: 1)
5. When photo provided with subject needing precise framing -> invoke photo-crop before writing HTML (from SKILL.md)
