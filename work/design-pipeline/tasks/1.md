# Task 1: Create shared design-references directory

## Required Skills

Перед выполнением задачи загрузи:
- `/skill:skill-master` — [skills/skill-master/SKILL.md](../../../skills/skill-master/SKILL.md)

## Description

Создаём директорию `shared/design-references/` и наполняем её 4 файлами дизайнерских знаний, которые будут использоваться несколькими design-скиллами (design-system-init, design-generate, design-review).

Три файла копируются из существующих источников: `design-tokens.md` (из `skills/like-figma/references/`), `color-psychology.md` и `color-principles.md` (из `work/design-pipeline/references/`). Четвёртый файл `library-catalog.md` создаётся с нуля — каталог бесплатных библиотек иконок и элементов со ссылками и описанием специфики.

Паттерн `shared/` уже существует в проекте (`shared/work-templates/`). Решение (Decision 1): shared-директория вместо дублирования или cross-skill ссылок. Решение (Decision 6): копирование файлов, не симлинки — скиллы должны быть self-contained, симлинки ненадёжны на Windows.

## What to do

1. Создать директорию `shared/design-references/`
2. Скопировать `skills/like-figma/references/design-tokens.md` в `shared/design-references/design-tokens.md` — без изменений содержимого
3. Скопировать `work/design-pipeline/references/color-psychology.md` в `shared/design-references/color-psychology.md` — без изменений содержимого
4. Скопировать `work/design-pipeline/references/color-principles.md` в `shared/design-references/color-principles.md` — без изменений содержимого
5. Создать `shared/design-references/library-catalog.md` — каталог бесплатных библиотек иконок и UI-элементов. Должен содержать: название библиотеки, ссылку, описание специфики, рекомендованные use-cases. Только ссылки и описания, без хранения ассетов
6. Проверить что все 4 файла существуют и непусты

## Details

**Files:**

- `shared/design-references/design-tokens.md` (new) — точная копия `skills/like-figma/references/design-tokens.md`. Содержит tokens.json schema (colors, typography, spacing, radii, shadows, breakpoints), naming conventions, CSS custom properties mapping. Примерно 80+ строк.
- `shared/design-references/color-psychology.md` (new) — точная копия `work/design-pipeline/references/color-psychology.md`. Содержит психологические профили 10 базовых цветов (белый, чёрный, красный, оранжевый, жёлтый, зелёный, синий, фиолетовый, коричневый, серый) и эмоциональную карту "эмоция -> сочетание".
- `shared/design-references/color-principles.md` (new) — точная копия `work/design-pipeline/references/color-principles.md`. Содержит 10 принципов: температурный контраст, 60-30-10 с инверсией, split complementary, монохромия с "предателем", оптическое смешение, grege-принцип, световой источник, асимметрия насыщенности, исторический колорит, simultaneous contrast.
- `shared/design-references/library-catalog.md` (new) — создать с нуля. Каталог бесплатных библиотек иконок и UI-элементов. Включить: Heroicons, Lucide, Phosphor Icons, Tabler Icons, Material Symbols, Radix Icons, Simple Icons, Feather Icons, а также библиотеки UI-элементов и иллюстраций. Для каждой: название, URL, лицензия, стиль, рекомендованные use-cases.
